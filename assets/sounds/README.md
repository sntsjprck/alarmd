# Alarm Sounds

Place alarm sound files here:
- `gentle.mp3` - Soft, pleasant wake-up tone
- `standard.mp3` - Default alarm sound
- `urgent.mp3` - Louder, more insistent tone

Search keywords for finding sounds:
- "gentle alarm chime"
- "soft wake up notification"
- "digital alarm tone"
- "morning wake up sound"

Recommended sources:
- freesound.org
